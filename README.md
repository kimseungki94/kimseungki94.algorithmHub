# backjoonHub 설명
- webHook 기능을 활용하여, 문제를 맞춘 후 소스파일(java파일), 설명(md파일)을 생성 후 푸쉬를 함 <br>

# 사용하게 된 이유
- 인텔리제이로 알고리즘 문제를 기존에 관리했었을 때, 클래스파일을 주기적으로 생성해야되는 점, <br>
문제번호는 적었지만 문제설명이 있지않아 항상 백준사이트에 직접들어가서 코드를 봐야하는 등의 어려움이 있었음
